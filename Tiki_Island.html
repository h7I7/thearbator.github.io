Tiki Island


GAME DOWNLOAD

Note This project was created with Unity version 5.6.1f1. The source code is not available for this project.

Controls

  • WASD - Move
  • R - Regenerate Maze

I created this project to demonstrate my ability with programming game play mechanics such as the dialogue boxes, interaction with items, and menus.

Most of the art was taken from the Unity asset store and is royalty free. The exception being the outline shader for interacting with items which I and some of my peers created. The understanding and basis for this shader comes from one of my other university modules, “Game Engine Programming” which goes into detail about how shaders work and how they are created.
